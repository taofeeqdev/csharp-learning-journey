# C# Learning Journey

An 18-month disciplined path from near-zero to .NET backend engineer on Azure.

**Start:** July 1, 2026  
**Target:** Remote role by December 2027  
**Commitment:** ~10 hours/week while working full-time  
**Location:** Nigeria → Remote (Gulf Coast priority)

---

## The Goal

Independently design, build, test, deploy, monitor, and maintain a production-grade .NET application on Azure.

Not a junior. A solid mid-level backend engineer with DevOps awareness.

---

## The 5 Non-Negotiable Rules

1. **NO AI for writing code.** Use it to explain concepts ONLY after 30+ minutes of struggle.
2. **NO copy-paste from tutorials.** Type every line. Understand every line.
3. **Every project from Month 5 onward deployed EXCLUSIVELY on Azure.** No exceptions.
4. **Every project from Month 8 onward has automated tests.** No exceptions.
5. **Write about your work publicly EVERY WEEK.** Remote employers hire people who communicate.

---

## The 4 Phases

| Phase | Duration | Focus | Deliverable |
|:---|:---|:---|:---|
| **Phase 1** | Jul–Oct 2026 (4 months) | C# Fundamentals + Git Mastery | Library Manager console app with clean Git history |
| **Phase 2** | Nov 2026–Mar 2027 (5 months) | ASP.NET Core + Azure-First Development | 3 web apps on Azure + live custom project |
| **Phase 3** | Apr–Aug 2027 (5 months) | Production-Grade Engineering | Multi-tenant SaaS capstone with full test coverage |
| **Phase 4** | Sep–Dec 2027 (4 months) | Remote Job Hunt | AZ-204 certification + remote job offer |
